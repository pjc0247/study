과제 2
----
* `setValue` 함수는 해당 함수를 다시 실행시키는 기능이 있습니다.

```tsx
const test = () => {
  const [a, setA] = useState(1);
  
  useEffect(() => {
    setTimeout(() => {
      setA(10);
    }, 1000);
  }, []);
  
  console.log(a);
};
```
* 위 코드에서 test는 총 2번 실행되어야 하며, 차례대로 `1`, `10` 이 출력되어야 합니다.
* 스스로 재실행하는 setter를 만들어보세요.
